# LevelUP — Детальный план реализации (Roadmap)

**Формат:** пошаговый план для одного fullstack-разработчика + lovable.dev
**Оценки:** в человеко-днях (1 день = 6–8 продуктивных часов)
**Ссылка на архитектуру:** [architecture.md](./architecture.md)

---

## Общая стратегия

**MVP-first подход:** максимально быстро запустить Платформу и Академию с полным функционалом критичных фич. Некритичные фичи — во второй итерации (Post-MVP). Один разработчик, вертикальные срезы: каждый спринт доставляет рабочую функциональность от БД до UI. lovable.dev — для быстрой генерации UI.

**Порядок:**
1. **Фаза 0** — Фундамент (общая инфраструктура)
2. **Фаза 1** — MVP Платформы (levelup-platform.ru)
3. **Фаза 2** — MVP Академии (levelup-academy.ru)
4. **Фаза 3** — Post-MVP: расширение Платформы и Академии
5. **Фаза 4** — Ассоциация (levelup-association.ru)
6. **Фаза 5** — Расширенные функции (AI, E2E-шифрование, магазин)

**Общий таймлайн до запуска MVP (Платформа + Академия):** ~6–7.5 месяцев

## Разделение фич: MVP vs. Post-MVP

### ПЛАТФОРМА

| MVP (полный функционал) | Post-MVP (Фаза 3) |
|---|---|
| Каталог коучей + поиск + фильтры | Виртуальная приёмная |

## Рекомендации для соло-разработчика

1. **lovable.dev на максимум в Фазах 0–2.** Генерируйте UI (формы, таблицы, карточки, дашборды). Экономия 30–40% на вёрстке.

2. **Фаза 3 — приоритизируйте по фидбеку.** После запуска MVP вы увидите, что реально нужно пользователям. Игровой движок может оказаться важнее кастомных доменов, или наоборот. Порядок задач в Фазе 3 — не жёсткий.

3. **Игровой движок (3.1.3–3.1.7) — самый рискованный блок.** 29 дней. tldraw + Three.js + Realtime — сложная связка. Начните с МАК-карт без 3D (2D-анимация), 3D-кубики — следующая итерация.

4. **Мультитенантность (2.1) — тестируйте тщательно.** Tenant Router, JWT, RLS — ошибка в изоляции данных = утечка данных школы. Выделите 2 дня на отдельное тестирование (2.7.2).

5. **Каждую неделю — рабочий деплой.** Это дисциплинирует и даёт раннюю обратную связь.

6. **Буфер 15%.** Реалистичный таймлайн до запуска MVP: **7.5–8.5 месяцев**.

7. **Split Payments — не блокирует запуск Академии.** На старте вся оплата идёт на платформу, вы рассчитываетесь со школами вручную. Split подключите, когда будет 10+ активных школ.

8. **Кастомные домены — не блокируют запуск.** Поддомены `*.levelup-academy.ru` дают каждой школе уникальный URL. Кастомные домены добавите в Post-MVP.
