# Чеклист тестовых сценариев — LevelUP Platform

## 1. Регистрация и авторизация
- [ ] Регистрация коуча (email + пароль) на https://levelup-platform.ru/auth?role=coach
- [ ] Регистрация клиента на https://levelup-platform.ru/auth
- [ ] Вход с существующим аккаунтом
- [ ] Выход из аккаунта
- [ ] Восстановление пароля через /auth/forgot-password

## 2. Профиль коуча
- [ ] Заполнение профиля: имя, фото, специализации, опыт, город
- [ ] Добавление описания/bio
- [ ] Установка почасовой ставки
- [ ] Загрузка документов для верификации
- [ ] Публикация профиля (is_published)
- [ ] Профиль отображается в каталоге /coaches

## 3. Каталог коучей
- [ ] Список коучей загружается на /coaches
- [ ] Поиск по ключевым словам
- [ ] Фильтры: специализация, город, цена, рейтинг
- [ ] Сортировка: по рейтингу, цене, отзывам
- [ ] Пагинация
- [ ] Клик на карточку → открывается профиль коуча /coach/:slug

## 4. Расписание и доступность
- [ ] Коуч настраивает расписание в /dashboard/schedule
- [ ] Установка рабочих дней и часов
- [ ] Слоты корректно отображаются на профиле коуча
- [ ] Блокировка дат (исключения)

## 5. Бронирование сессии
- [ ] Клиент выбирает дату и время на профиле коуча
- [ ] Создание бронирования через API
- [ ] Бронирование появляется в дашборде коуча
- [ ] Бронирование появляется в дашборде клиента
- [ ] Уведомление о новом бронировании

## 6. Видеосессия
- [ ] Кнопка "Начать сессию" появляется за 5 минут до начала
- [ ] Видеокомната открывается (/dashboard/video/:sessionId)
- [ ] Камера и микрофон работают
- [ ] Оба участника видят друг друга
- [ ] Завершение сессии

## 7. Мессенджер
- [ ] Открытие чата из /dashboard/messages
- [ ] Отправка текстового сообщения
- [ ] Получение сообщения в реальном времени
- [ ] Список бесед загружается

## 8. Заметки и отзывы
- [ ] Коуч может оставить заметку после сессии
- [ ] Клиент может оставить отзыв и оценку
- [ ] Отзывы отображаются на публичном профиле коуча
- [ ] Средний рейтинг обновляется

## 9. Дашборд
- [ ] Дашборд коуча: статистика, ближайшие сессии
- [ ] Дашборд клиента: предстоящие сессии
- [ ] Уведомления отображаются
- [ ] Настройки профиля доступны

## 10. Финансы
- [ ] Страница финансов коуча загружается
- [ ] История транзакций отображается

## 11. Мобильная версия
- [ ] Лендинг корректно отображается на мобильном
- [ ] Каталог адаптивен
- [ ] Дашборд адаптивен (мобильное меню)
- [ ] Формы удобны на мобильном

## 12. SEO и метаданные
- [ ] Title и description на всех страницах
- [ ] sitemap.xml доступен
- [ ] robots.txt доступен
- [ ] Open Graph теги корректны

## Инструкция для тестировщика

**URL**: https://levelup-platform.ru
**API**: https://api.levelup-platform.ru

### Учетные данные для тестирования
- **Тестовый аккаунт коуча**: создайте новый через /auth?role=coach
- **Тестовый аккаунт клиента**: создайте новый через /auth

### Процедура регистрации бага

При обнаружении ошибки, пожалуйста, опишите:

1. **Шаги воспроизведения** — последовательность действий, которые привели к ошибке
2. **Ожидаемый результат** — что должно было произойти
3. **Фактический результат** — что произошло вместо этого
4. **Скриншот** — если возможно, приложите визуальное подтверждение
5. **Браузер и устройство** — какой браузер и ОС используется для тестирования
